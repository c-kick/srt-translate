# Register Guide (je/u)

Single source of truth for Dutch T-V pronoun choices in subtitle translation.

---

## Core rule: je is the default

Dutch subtitling uses *je/jij* unless there is a clear reason for *u*. When in doubt, use *je*.

## Always je/jij

| Relationship | Reason |
|---|---|
| Spouses, partners | Intimate relationship — NEVER u |
| Family members (siblings, parent↔child) | Family bond |
| Close friends | Familiar relationship |
| Colleagues at same level | Peer relationship |
| Staff among themselves | Informal workplace |
| Boss → employee | Downward hierarchy |
| Adult → child | Age difference (downward) |

## Always u/uw

| Relationship | Reason |
|---|---|
| Service provider → customer/guest | Professional courtesy |
| Employee → unfamiliar superior | Upward hierarchy |
| Strangers in formal setting | Social distance |
| Addressing elderly (by younger stranger) | Respect |
| Professional titles (doctor, lawyer) | Formal role |

## Context-dependent

| Relationship | Guidance |
|---|---|
| Employee → familiar boss | Match source register — may be je if relationship is informal |
| Strangers in casual setting | Often starts u, may shift to je |
| Child → parent | Usually je in modern Dutch; u only if period piece or very formal family |

## Register shifts

Speakers may shift register within a scene:
- Strangers meeting → start with u, shift to je as rapport builds
- Anger or confrontation → formal u can break into je (emotional slip)
- A character using u sarcastically signals cold distance

Mark register shifts in the batch context so they carry across batches.

## Consistency rules

- **Track T-V per character pair**, not per character. A may use je with B but u with C.
- Record every T-V decision in the batch context under "Register (T-V per speaker)".
- Switching je/u for the same character pair without a dramatic reason is a continuity error.
- When resuming from a previous batch or invocation, read the glossary and batch context for established register choices before translating.

## Genre overrides

Genre translator files may override these base rules. When they do, the override is explicitly labelled. Check your genre translator for any exceptions before applying the base rules.
