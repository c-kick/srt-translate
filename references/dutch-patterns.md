# Dutch Patterns Reference

## V2 Word Order (Critical)

Dutch is a **verb-second (V2) language**: the finite verb must occupy the second position in main clauses. This is the most common source of anglicisms in translation.

### Inversion After Fronted Elements

When ANY element other than the subject starts the sentence, **invert subject and verb**:

| English | ✗ Incorrect (anglicism) | ✓ Correct Dutch |
|---------|------------------------|-----------------|
| "Still... we are at war" | "Nog steeds... wij zijn in oorlog" | "Nog steeds... zijn wij in oorlog" |
| "Then he left" | "Toen hij vertrok" | "Toen vertrok hij" |
| "Now I understand" | "Nu ik begrijp het" | "Nu begrijp ik het" |
| "Yesterday we went" | "Gisteren we gingen" | "Gisteren gingen we" |
| "Therefore I believe" | "Daarom ik geloof" | "Daarom geloof ik" |
| "Here comes the train" | "Hier de trein komt" | "Hier komt de trein" |

### Common Fronted Elements

Watch for these at sentence start — they ALL trigger inversion:

| Type | Examples |
|------|----------|
| Time adverbs | nu, toen, daarna, gisteren, vandaag, straks, ooit |
| Place adverbs | hier, daar, overal, nergens |
| Manner adverbs | zo, dus, daarom, daardoor, toch |
| Conjunctive adverbs | echter, bovendien, namelijk, immers |
| Prepositional phrases | "In 1944...", "Na de oorlog...", "Op dat moment..." |
| Objects (for emphasis) | "Dit boek heb ik gelezen" (not "Dit boek ik heb gelezen") |

### After Ellipsis

Ellipsis (...) does NOT reset word order. If the fragment before the ellipsis would trigger inversion, apply it:

```
EN: "Still... we are at war."
NL: "Nog steeds... zijn wij in oorlog."
    ^^^^^^^^      ^^^^^^^^
    fronted       verb before subject
```

### Subordinate Clauses

After a subordinate clause, the main clause verb comes FIRST:

| English | ✗ Incorrect | ✓ Correct |
|---------|------------|-----------|
| "When he arrived, he saw..." | "Toen hij aankwam, hij zag..." | "Toen hij aankwam, zag hij..." |
| "If that's true, we must..." | "Als dat waar is, we moeten..." | "Als dat waar is, moeten we..." |

### Questions

Questions also require inversion (verb before subject):

| English | Dutch |
|---------|-------|
| "Do you understand?" | "Begrijp je het?" |
| "What did he say?" | "Wat zei hij?" |

## Translation Patterns

| English | Bad Dutch (literal) | Good Dutch (natural) |
|---------|--------------------|--------------------|
| "I'm going to..." | "Ik ga..." | "Ik zal..." / "Ik ga zo..." |
| "He said that..." | "Hij zei dat..." | Often restructure entirely |
| "It's raining cats and dogs" | *Never translate literally* | "Het regent pijpenstelen" |
| "You know what I mean?" | "Je weet wat ik bedoel?" | "Snap je?" / "Toch?" |
| "And, you know, it felt like..." | "En, weet je, het voelde als..." | "Het voelde als..." |
| "Well, I didn't want to..." | "Nou, ik wilde niet..." | "Ik wilde niet..." |

## Dutch Directness

Dutch subtitles favor directness over English rhetorical structures.

### Drop filler phrases (unless they carry emotional weight):

| English filler | Action |
|---------------|--------|
| "Well, ..." | Usually omit |
| "You know, ..." | Usually omit |
| "I mean, ..." | Usually omit |
| "Oh." (hesitation) | Omit unless dramatic |
| "And, uh, ..." | Restructure without |

### Convert rhetoric to direct statements:

| English (rhetorical) | Dutch (direct) |
|---------------------|----------------|
| "What do you think I am, an idiot?" | "Ben ik een idioot?" |
| "Isn't that cutting it close?" | "Dat is kort dag." |
| "Don't you think we should...?" | "We moeten..." |
| Tag questions ("...right?", "...isn't it?") | Usually omit |

### Active over passive in dialogue:

| English (passive/formal) | Dutch (active/conversational) |
|-------------------------|------------------------------|
| "The agreement includes payments" | "Ik krijg betalingen" |
| "It was decided that..." | "Ze besloten..." |

## Dutch Idioms

| English expression | Dutch idiom | Avoid (literal) |
|-------------------|-------------|-----------------|
| "couldn't believe it" | "geloofde z'n oren/ogen niet" | "kon het niet geloven" |
| "making me look bad" | "zet me te kakken" / "zet me voor schut" | "laat me slecht lijken" |
| "go to hell" | "bekijk het maar" / "bekijk 't" | "ga naar de hel" |
| "cutting it close" | "kort dag" | "dat is krap" |
| "takes the cake" | "slaat alles" / "spant de kroon" | — |
| "borderline" (mental) | "van slag" | "op het randje" |
| "crossed the line" | "doorgeslagen" | "ging erover" |
| "give me a hand" | "help 's even" | "help me even" |

**When English uses wordplay, seek Dutch wordplay:**
- EN: "He was borderline... then he crossed the border"
- NL: "Hij was van slag... toen is hij doorgeslagen"

## Contractions

Use contractions sparingly. Only two are standard in subtitles:

| Full form | Contraction | When to use |
|-----------|-------------|-------------|
| mijn | m'n | Informal speech (unstressed) |
| zijn (possessive) | z'n | Informal speech (unstressed) |

**Avoid these in subtitles** (per Auteursbond guidelines):

| Contraction | Use instead |
|-------------|-------------|
| 't | het |
| 's | eens |
| 'n | een |
| d'r | haar |
| 'm | hem |

**Exception:** Set phrases where the contraction is standard: "'s ochtends", "'s avonds", "'s nachts".

**Example:**
- Good: "Heb je m'n beker gezien?" ✓
- Good: "Dat is z'n probleem." ✓
- Avoid: "'t Was mooi." → "Het was mooi." ✓
- Avoid: "Kijk 's." → "Kijk eens." ✓

## Register Guidance (je/u)

| Context | Register | Example |
|---------|----------|---------|
| Casual dialogue | je/jij | "Heb je dat gezien?" |
| Formal settings | u/uw | "Hebt u dat gezien?" |
| Child to parent | je (usually) | Context-dependent |
| Employee to boss | Varies by scene | Match source register |
| Strangers | Often u initially | May shift to je |

**Maintain T-V consistency** per character relationship throughout.

## Bottom-Heavy Pyramid

For two-line cues, shorter line on top:

```
✓ Korte eerste regel
  en langere tweede regel hier.

✗ Dit is de langere eerste regel
  Korte tweede.
```

## Quality Benchmarks (from Gold Standards)

Professional Dutch subtitles share these characteristics:

- **Condensation without loss**: "We vroegen wat hij bedoelde" not "Wij vroegen hem wat hij daarmee bedoelde"
- **Natural contractions**: "z'n", "d'r", "'t was", "m'n"
- **Proper register shifts**: Formal narration vs. casual dialogue
- **Invisible transitions**: Sentence flow feels continuous across cues
- **Cultural adaptation**: Metric system, Dutch idioms, local references
- **Merged cues**: 65-80% of source count, 8-15 word reading units
- **Dutch directness**: Rhetoric and filler removed, meaning preserved
- **Genre-appropriate vocabulary**: Military, scientific, or colloquial as needed
- **No visual redundancy**: Omit greetings/acknowledgments visible on screen
- **Comedy timing**: Shorter beats, single-word punches, brevity over completeness
- **Rapid-fire collapse**: Multiple quick interjections → single essential message

## Common Anglicisms to Avoid

English loanwords that look Dutch but aren't:

| Anglicism (avoid) | Proper Dutch |
|-------------------|--------------|
| counteroffensief | tegenoffensief |
| committeren | vastleggen / toewijzen |
| downloaden | downloaden (accepted) / ophalen |
| updaten | bijwerken |
| cancelen | annuleren / afzeggen |
| checken | controleren / nakijken |
| deleten | verwijderen / wissen |
| managen | beheren / leiden |
| performen | presteren |
| schedulen | plannen / inroosteren |
| shippen | verschepen / verzenden |
| troubleshooten | problemen oplossen |

**Rule of thumb:** If an English word has a -en suffix added to make it "Dutch," there's usually a proper Dutch equivalent. Technical terms (downloaden) are more accepted than business jargon (committeren).
